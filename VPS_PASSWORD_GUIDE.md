# VPSパスワード確認ガイド

## 🔐 パスワードの確認方法

### 方法1: XサーバーVPSパネルで確認（推奨）

1. **XサーバーVPSパネルにログイン**
   - URL: https://secure.xserver.ne.jp/xvps_login.php
   - Xserverアカウントでログイン

2. **VPS管理画面を開く**
   - 契約一覧から「VPS」タブを選択
   - サーバー名「**oriental**」をクリック

3. **パスワードを確認/変更**
   - 左メニューから「**パスワード変更**」をクリック
   - 現在のパスワードが表示される、または
   - 新しいパスワードを設定できる

### 方法2: 契約時のメールを確認

契約時に送信されたメールを確認してください：

**件名**: `【Xserver VPS】■重要■VPSアカウント設定完了のお知らせ`

このメールに初期パスワードが記載されています。

### 方法3: パスワードを再設定

パスワードが分からない場合、XサーバーVPSパネルで再設定できます：

1. XサーバーVPSパネルにログイン
2. VPS管理画面で「パスワード変更」をクリック
3. 新しいパスワードを設定（8文字以上、英数字記号を含む）

## 📱 シリアルコンソール（VNCコンソール）へのアクセス

### 手順

1. **XサーバーVPSパネル**にログイン
   - https://secure.xserver.ne.jp/xvps_login.php

2. **VPS管理画面**を開く
   - 「VPS」タブをクリック
   - サーバー「**oriental**」を選択

3. **コンソールを起動**
   - 「**コンソール**」または「**VNCコンソール**」ボタンをクリック
   - ブラウザ上でターミナルが開きます

4. **ログイン**
   - ユーザー名: `root`
   - パスワード: XサーバーVPSパネルで確認したパスワード

## 🔑 パスワードの要件

XサーバーVPSのパスワード要件：
- **8文字以上**
- **英数字記号を含む**
- 推奨: 大文字・小文字・数字・記号を組み合わせる

## 💡 パスワードが見つからない場合

### 1. メールを検索

メールクライアントで以下を検索：
- 送信者: `Xserver` または `xserver`
- 件名: `VPS` または `アカウント設定完了`

### 2. XサーバーVPSパネルで再設定

パスワードが分からない場合は、XサーバーVPSパネルで再設定してください。

### 3. Xサーバーサポートに問い合わせ

上記の方法で解決しない場合：
- **サポートページ**: https://www.xserver.ne.jp/support/
- **サポートチケット**: VPSカテゴリで問い合わせ

## 🚀 パスワード確認後の作業

パスワードが確認できたら、以下でSSH接続を試してください：

```bash
ssh root@162.43.15.173
```

接続できたら、ログインエラーの修正作業を進められます。
