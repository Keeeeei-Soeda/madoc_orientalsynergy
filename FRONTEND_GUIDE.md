## 🚀 起動方法

### 前提条件

Node.js 18 以上がインストールされていること

### ステップ 1: 依存パッケージのインストール

```bash
cd /Users/soedakei/madoc_line/frontend
npm install
```

### ステップ 2: 環境変数の設定（オプション）

```bash
cp env.local.example .env.local
# .env.local ファイルを必要に応じて編集
```

### ステップ 3: 開発サーバーの起動

```bash
npm run dev
```

### ステップ 4: ブラウザでアクセス

```
http://localhost:3000
```

自動的に管理者ダッシュボードへリダイレクトされます。

## 💡 ヒント

### スタイリング

- Bootstrap クラスを優先的に使用
- カスタムスタイルは `globals.scss` に追加
- 変数は `variables.scss` で管理

### コンポーネント

- 'use client' は必要な場合のみ（useState, useEffect 等を使う場合）
- Server Component がデフォルト
- 型定義を忘れずに

### ルーティング

- Next.js App Router を使用
- ファイル名 `page.tsx` がルートになる
- 動的ルート: `[id]/page.tsx`
