### **推奨 B: 自動マッチングのみ削除**

```

- 月間予約数が少ない初期段階では不要

- 後から追加も可能（+¥250,000程度）
- LINEは通知の即時性に重要なので残す
```

### **推奨 C: 自動マッチング + LINE 両方削除**

```
費用: ¥5,360,000
期間: 約7.5ヶ月
削減: -¥400,000

理由:
- 予算を最優先する場合
- スタッフが少数で管理しやすい
- メール通知でも運用可能な環境
```
