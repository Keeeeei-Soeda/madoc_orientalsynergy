## 🎯 ¥200 万版で実現できること

### 実際の運用イメージ

#### 1 日の流れ

```
【管理者】
09:00 予約が入る
09:10 スタッフを検索・絞り込みしてアサイン
09:15 メールでスタッフに通知が届く

【スタッフ】
09:20 メールを確認
09:25 システムにログインして受諾
10:00 業務開始、出勤打刻
14:00 業務完了、退勤打刻・完了報告

【企業】
14:30 完了報告を確認
14:35 評価を入力（5段階+コメント）

【スタッフ】
15:00 評価を受信（メール通知）
15:05 システムで評価を確認
```

### 月次の流れ

```
【スタッフ】
毎月25日 シフト希望を提出

【管理者】
毎月28日 シフトを承認
毎月月末 勤怠データを確認（自動集計）

【企業】
毎月5日 月次レポートを確認
```
