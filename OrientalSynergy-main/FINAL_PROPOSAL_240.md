## 🚀 新機能: LINE 打刻システム

### リッチメニュー

LINE 画面下部に常時表示されるメニュー

```
┌──────────┬──────────┐
│ 📅       │ 🔔       │
│ 今日の   │ 通知     │
│ 業務     │          │
├──────────┼──────────┤
│ 🟢       │ 🔴       │
│ 出勤     │ 退勤     │ ← タップで即打刻！
│ 打刻     │ 打刻     │
├──────────┼──────────┤
│ 📊       │ 📱       │
│ 今月の   │ Web      │
│ 実績     │ アプリ   │
└──────────┴──────────┘
```

### 出勤打刻の流れ（わずか 10 秒！）

```
1. 現場到着
   ↓
2. LINEアプリを開く
   ↓
3. 画面下の[出勤打刻]をタップ
   ↓
4. LINE内で打刻画面が開く（LIFF）
   ・時刻: 10:00:23（自動取得）
   ・写真を撮る（任意）
   ・メモ入力（任意）
   ↓
5. [打刻する]をタップ
   ↓
6. 完了！「出勤を記録しました」

所要時間: 約10秒
```

### 退勤打刻の流れ（約 30 秒）

```
1. 作業完了
   ↓
2. LINEアプリを開く
   ↓
3. [退勤打刻]をタップ
   ↓
4. LINE内で打刻画面が開く
   ・出勤時刻: 10:00
   ・勤務時間: 4時間（自動計算）
   ・休憩時間を入力
   ・作業数を入力
   ・完了写真を撮影（任意）
   ・作業内容を記入（任意）
   ↓
5. [打刻する]をタップ
   ↓
6. 完了！「退勤を記録しました」

所要時間: 約30秒
```

## 💡 システム構成

### フロントエンド

- **技術**: Next.js 14 + TypeScript
- **UI**: Bootstrap 5（既存を活用）+ カスタム CSS
- **対応**: レスポンシブデザイン（PC・タブレット・スマホ）
- **LIFF**: LINE Front-end Framework（打刻部分のみ）

### バックエンド

- **技術**: FastAPI (Python 3.11+)
- **API**: RESTful API（OpenAPI 仕様書付き）
- **認証**: JWT + OAuth2.0

### データベース

- **DB**: PostgreSQL 15
- **キャッシュ**: Redis（セッション管理）

### インフラ

- **環境**: AWS
  - EC2（アプリケーションサーバー）
  - RDS（データベース）
  - S3（画像・ファイルストレージ）
  - CloudFront（CDN）
- **SSL**: Let's Encrypt（無料 SSL 証明書）

### 外部連携

- **LINE**: LINE Messaging API + LIFF
- **メール**: SendGrid または AWS SES
- **地図**: Google Maps API（予約時の地図表示のみ）
